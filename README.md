# 🔌 Raspberry Pi Relay Control

Control 3 relays using GPIO pins on Raspberry Pi

## 📋 Description
This project controls 3 relays connected to Raspberry Pi 
using GPIO pins. Relays turn ON for 3 seconds and then 
turn OFF automatically.

## 🛠️ Hardware Required
- Raspberry Pi
- 3 Channel Relay Module
- Jumper Wires

## 📌 GPIO Pin Configuration
| Relay | GPIO Pin |
|-------|----------|
| Relay 1 | GPIO 17 |
| Relay 2 | GPIO 27 |
| Relay 3 | GPIO 22 |

## 💻 Software Required
- Python 3
- RPi.GPIO library

## ▶️ How to Run
```bash
python3 relay_control.py
```
